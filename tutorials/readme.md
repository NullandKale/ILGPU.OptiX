# ILGPU.Optix

ILGPU.Optix is a port of [NVIDIA OptiX 9.0.0 SDK](https://developer.nvidia.com/designworks/optix/downloads/legacy) for ILGPU. It enables the use of the NVIDIA OptiX high-performance ray-tracing API inside of C#. No longer are you shackled to writing cumbersome c++ to write the fastest best-est ray tracing code in the world. 

This repo is AI converted the Optix 9.0.0 SDK to ILGPU by porting SDK code and the samples from [Siggraph 2019/2020 OptiX 7 (and newer) Course Tutorial Code](https://github.com/ingowald/optix7course). 

Any of the markdown files in this repo are hand written by me @NullandKale. I don't think its super fair to ask someone to read AI generated documentation but the downside to that is you have to contend with my spelling and grammar mistakes. If you want AI generated documentation clone the repo and ask your agent questions.
